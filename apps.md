# Manually Installed Apps

These apps are NOT managed by Homebrew and need manual installation on a new machine.

## Productivity
- **Alfred 5** — launcher and productivity tool (alfredapp.com)
- **Caffeinated** — prevent Mac from sleeping

## Communication
- **LarkSuite** — team communication
- **Slack** — team communication
- **Spark Desktop** — email client

## Browsers
- **Firefox** — web browser
- **Google Chrome** — web browser

## Development
- **Cursor** — AI code editor (cursor.com)
- **Docker Desktop** — container runtime (docker.com)
- **Fork** — git GUI client (git-fork.com)
- **Ghostty** — terminal emulator (ghostty.org)
- **Warp** — terminal emulator (warp.dev)
- **Xcode** — Apple developer tools (via App Store)

## AI
- **ChatGPT** — OpenAI assistant (via Mac App Store or openai.com)
- **Claude** — Anthropic assistant (installed via Homebrew cask)

## Creative
- **Pixelmator Pro** — image editing (via App Store)

## Notes
- **Bear** — markdown notes (via App Store)
